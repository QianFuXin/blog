# mac的Chrome自动化
## 配置驱动

```shell
brew install chromedriver
```

## 开启Chrome

[start-chrome-debug.sh](https://gist.github.com/QianFuXin/545636e829e2e587e5e5b69bf97be6c7)

```shell
chmod +x start-chrome-debug.sh
./start-chrome-debug.sh
```

pip install selenium
